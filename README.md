## Featured project

### [PlateGuardian / DetecPlat](https://github.com/ratuqra/DetecPlat)

A desktop application that detects and reads Indonesian vehicle license plates using image processing and template matching.

- Detects plate regions with edge detection, morphology, and contour analysis
- Segments and recognizes characters using 116 alphanumeric templates
- Visualizes each image-processing stage in an interactive PyQt5 interface
- Evaluates both character-level and full-plate recognition accuracy

`Python` · `OpenCV` · `PyQt5` · `NumPy` · `Computer Vision`
